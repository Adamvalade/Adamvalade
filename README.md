## Projects

- C++ engine logic exposed to Python

**Tech:** C++, Python, JavaScript, Node.js

### ForgeQueue
A Redis-backed job queue built to handle failures cleanly.

- Worker heartbeats and crash recovery
- Retry logic with exponential backoff
- Dead-letter queue support
- Duplicate job prevention

**Tech:** Python, Redis, PostgreSQL, Docker

### Cadence
A music logging and review app.

- User reviews, lists, and profiles
- Authentication and database-backed accounts
- Full-stack app with PostgreSQL and Redis

**Tech:** FastAPI, Next.js, TypeScript, PostgreSQL, Redis
